# Quick Notes Pro

## Overview

Quick Notes Pro is a modern note-taking web application built using Flask, SQLite, HTML, CSS, and JavaScript. It allows users to create, save, search, and manage notes through an interactive and responsive interface.

## Features

* Create Notes
* Save Notes to SQLite Database
* Search Notes
* Edit Notes
* Delete Notes
* Responsive User Interface
* Flask Backend Integration
* SQLite Database Storage

## Tech Stack

### Frontend

* HTML5
* CSS3
* JavaScript

### Backend

* Flask
* Flask-CORS

### Database

* SQLite

## Project Structure

backend/

* app.py
* notes.db
* requirements.txt

frontend/

* index.html
* style.css
* script.js

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Start Flask server

python app.py

3. Open index.html in browser
